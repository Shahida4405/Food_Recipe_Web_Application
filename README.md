# Food Recipe Web Application

A MERN Stack Food Recipe Web Application that allows users to:

- Register and Login
- Search Recipes
- View Recipe Categories
- Save Favorite Recipes
- Manage User Profiles
- Add and Manage Recipes (Admin)

## Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Installation

git clone https://github.com/Shahida4405/Food_Recipe_Web_Application.git

cd Food_Recipe_Web_Application

npm install

npm start
